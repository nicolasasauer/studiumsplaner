# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:

- Being respectful and constructive
- Accepting feedback gracefully
- Focusing on what is best for the community

Unacceptable behavior includes:

- Harassment, insults, or discriminatory language
- Trolling, personal attacks, or intimidation
- Publishing private information without permission

## Enforcement

Project maintainers are responsible for clarifying and enforcing standards of acceptable behavior.

## Scope

This Code of Conduct applies within project spaces and in public spaces when representing the project.
